# 酒に溺れる人
酒に溺れて吐く人です

## 説明
1. マウスの位置によって、ウイスキーの瓶が傾きます。
1. ウイスキーの瓶が傾くとウイスキーが出ます。
1. ウイスキーを飲みすぎると、目を回します
1. 飲みすぎると目が高速に回転し吐きます
